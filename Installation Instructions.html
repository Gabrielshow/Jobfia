Jobfia Installation Instructions

  1. Unzip the package in an empty directory

  2. Open /upload/inc/config.inc.php and fill in your MySQL database settings

  3. Upload all the files and folders from the "Upload" folder to your public_html (or www) directory

  4. If you have a Linux/Unix make sure 'uploads', 'images/avatars', 'images/offers' folders are writable. Chmod 0755 or 0777

  5. Open in the browser: http://yourdomain.com/install.php

  6. Run Installation

  7. Note your admin area password

  8. Delete the 'install.php' file from your server!

Enjoy using Jobfia!
